Отримання SSL-сертифікату (Let’s Encrypt)

Встанови Certbot та плагін для Nginx:
sudo apt update
sudo apt install certbot python3-certbot-nginx -y

Налаштуй домен (обов’язково, потрібен реальний домен, наприклад example.com)
На DNS-домені зроби A-запис на зовнішній IP твого Netis WF2780 - 195.158.196.129

На роутері Netis WF2780 відкрий порти:
80 TCP → 192.168.1.100 (HTTP, для верифікації Certbot)
443 TCP → 192.168.1.100 (HTTPS)

Перевір, що домен доступний через HTTP
Переконайся, що http://piano.in.ua відкривається.
Nginx вже обслуговує сайт із /var/www/piano.in.ua.

Генерація SSL сертифікату та автоматична настройка Nginx
sudo certbot --nginx -d piano.in.ua -d www.piano.in.ua

Перевір роботу HTTPS
https://piano.in.ua

Автооновлення сертифікатів
Сertbot автоматично додає cron для оновлення сертифікату.
sudo certbot renew --dry-run

Примусове HTTPS
Certbot зазвичай пропонує автоматичне перенаправлення HTTP → HTTPS.
Якщо ні, додай у конфігурацію Nginx:
server {
    listen 80;
    server_name example.com;
    return 301 https://$host$request_uri;
}

Перезавантаж Nginx:
sudo systemctl reload nginx

Додаткові налаштування безпеки
SSH через ключі, змінити стандартний порт 22.

Регулярні оновлення:
sudo apt update && sudo apt upgrade -y
